# Console Sales Analysis
授業課題をベースに、家庭用ゲーム機（PlayStation、Wii、3DS）の販売本数データを集計・可視化したプログラムです。  
年×プラットフォーム別に販売本数を整理し、推移を折れ線グラフで表示しました。

## 使用技術
- Python（pandas、matplotlib）

## 工夫点
- 上位プラットフォームに絞った集計処理
- 各プラットフォームを色分けし、比較しやすい可視化を意識
